A linguagem HTML

O HTML (HyperText Markup Language) é a linguagem de marcação padrão para criar páginas da web.
Ele foi desenvolvido por Tim Berners-Lee em 1991 e desde então passou por várias versões, cada uma adicionando novos recursos e melhorias.

Alguns dos recuros da linguagem HTML vistos até o momento:

Recurso Tag Descrição
Links <a> Elementos básicos para criar hyperlinks
Atributos <tag atributo="valor"> Propriedades adicionais para elementos HTML
Listas <ul> e <li> Organização de itens não ordenados
<ol> e <li> Organização de itens ordenados
Tabelas <table> Estruturação de dados em linhas e colunas
<tr> Definição de uma linha na tabela
<td> Definição de uma célula na tabela
<th> Definição de uma célula de cabeçalho na tabela
Imagens <img> Incorporação de imagens em páginas web
